# FloorRush

A simple 3D endless runner game built with Unity and C#. The player continuously moves forward and must dodge incoming obstacles to survive as long as possible.

## Features

* Endless forward movement
* Obstacle spawning system
* Player movement and lane switching
* Collision detection and game-over system
* Score tracking based on survival time/distance
* Physics-based gameplay using Unity Rigidbody

## Technologies Used

* Unity Game Engine
* C#
* Unity Physics System (Rigidbody, Colliders)

## Gameplay

The player automatically moves forward on an endless path while obstacles appear ahead. The objective is to avoid collisions and survive for as long as possible to achieve a high score.

## Controls

* A / Left Arrow: Move Left
* D / Right Arrow: Move Right
* Space: Jump (if implemented)

## Project Structure

* `PlayerMovement.cs` – Handles player movement and controls
* `ObstacleSpawner.cs` – Spawns obstacles at regular intervals
* `GameManager.cs` – Manages score and game state
* `UIManager.cs` – Handles score display and game-over screen
